Step 1: Identify the coefficients:

Here is the base form of a quadratic equation:

  • The quadratic equation: ax^2 + bx + c = 0
  • Example: x^2 - 6x + 8 = 0
  • Coefficients: a = 1, b = -6, c = 8

Step 2: Calculate the Discriminant (b^2 - 4ac):

Let's break this down into chunks:

  • b^2:
    • ^2- Square the number. This mean you must multiply a number by itself.
    • -6^2
    • -6 x -6
    • Result: 36 The first half of the discriminant (b^2) is solved!
  • 4ac:
    • Calculation: 4 x 1 x 8 = 32
    • The second half (4ac) is solved! Now, we subtract the result of 4ac from the result of b^2 to find the discriminant:
    • 36 - 32 = 4 Thus, 4 is our discriminant, which will be used under the square root in the quadratic formula.

Step 3: Use the quadratic formula:

  • Formula: x = (-b ± √(Discriminant)) / (2a)
  • Substitute the values: x = (-(-6) ± √4) / (2 * 1) Here, we see how the discriminant (calculated in Step 2) fits into the quadratic formula under the square root.

Step 4: Simplify the equation:

  • -(-6) = 6
  • 2 x 1 = 2
  • √ - This means that we are trying to figure out which number multiplied by itself will equal the number attached to the root.
  • √4 = 2 [This is because 2 x 2 = 4, so 2 is the number that when multiplied by itself is equal to 4 which was attached to the root.]
  • Simplified Equation: x = (6 ± 2) / 2 This step-by-step simplification clearly shows how the value from the square root influences the solutions.

Step 5: Solve for x:

  • Note: ± - Splits the equation into 2 different equations. One will have a plus sign in the parenthesis, while the other has a minus sign.
  • Solve x1: x1 = (6 + 2) / 2
  • Solve x1: x1 = 8 / 2
  • Solve x1: x1 = 4
  • Solve x2: x2 = (6 - 2) / 2
  • Solve x2: x2 = 4 / 2
  • Solve x2: x2 = 2

So, the solutions to the quadratic equation x^2 - 6x + 8 = 0 are x = 4 and x = 2.
